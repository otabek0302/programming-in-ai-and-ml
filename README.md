# Programming in AI & ML

A structured Python learning resource — annotated, runnable `.py` files organized by topic and difficulty.

## Structure

```
python-basics/       # Core Python concepts (01–31 topics)
python-advanced/     # Advanced Python topics (01–15 topics)
python-tasks/        # Practice exercises (task-1 through task-20)
```

## How to Use

1. Browse a topic file and read the comments to learn the concept.
2. Open a task file, read the problem description, and write your solution below each task.
3. Run any file directly: `python3 python-basics/08_strings/strings.py`

## Requirements

- Python 3.10+
- `numpy` for the arrays topic: `pip install numpy`
- `pytest` for the testing topic: `pip install pytest`
